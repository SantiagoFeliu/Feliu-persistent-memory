# Lessons Learned

## Memory Quality Matters More Than Memory Size

The primary challenge is not storing more information.

The primary challenge is preserving useful information.

## Facts And Inferences Must Be Separated

A persistent memory system should distinguish:

- verified facts
- inferred facts
- historical decisions
- active rules
- obsolete rules

## RAG Is Not Enough

Semantic retrieval solves context discovery.

Relationship discovery requires an additional layer.

Knowledge Graphs complement RAG.

## Historical Data Requires Auditing

Large conversation archives contain:

- contradictions
- outdated assumptions
- duplicated concepts
- obsolete decisions

Memory quality improves through auditing.

## Regeneration Is Better Than Manual Maintenance

Persistent memory systems should be reproducible.

The preferred model is:

source data
→ extraction
→ regeneration

instead of manual editing.

## Local Operation Simplifies Governance

Keeping memory local reduces:

- privacy risks
- dependency on providers
- vendor lock-in

while increasing control.
